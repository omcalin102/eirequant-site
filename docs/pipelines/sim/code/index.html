Ablation checklist

  • Remove sector block
  • Remove volatility block
  • Alter horizon h
  • Swap policy gains (Kp/Ki/Kd)
  • Clamp variants and turnover caps

Diagnostics to track

  • ECE before/after scaling
  • Regime stability deltas
  • Slippage & turnover
  • Outlier sensitivity
